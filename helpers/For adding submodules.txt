STEPS

step1: create your pkg or folder in src, then push it to github
cd ~/ros2_ws/src/FOLDER
git init
git add .
git commit -m "Initial commit"
git remote add origin <your-FOLDER-repo-url>
git push -u origin master

step2: remove the folder from your workspace
rm -rf ~/ros2_ws/src/FOLDER

step3: add the folder as a submodule in src
cd ~/ros2_ws/src
git submodule add <your-FOLDER-repo-url>
